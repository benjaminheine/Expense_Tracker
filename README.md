## Installation Instruction
1. In the root directory, run `npm i`
2. In the root directory, run `npm run start`

## Deliverables
Each expense should include:
- The date
- A description
- The amount
- Add a new expense
- Delete expense
- All of the expenses shown up in an HTML table
